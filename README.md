# Maps 🗺️

<strong><p>Projeto feito em TypeScript usando React Native e Expo.</p></strong>

<p>Bibliotecas usadas no projeto:</p>
<ul>
  <li>react-native-maps</li>
  <li>expo-location</li>
</ul>
